# message-board
## message-board - series of learning react js from React Quickly book
### system dependencies
* mongodb(db: board, collections: messages)
### how to start
1. git clone <https://github.com/MusaAbdillah/message-board.git>
2. cd message-board
3. npm install
4. npm run build
5. npm start!
6. navigate browser to localhost:3000
7. enjoy
